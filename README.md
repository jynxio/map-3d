# 概述

该项目是使用 [three.js](https://www.npmjs.com/package/three) 来实现的小小 3d 地图，原意是要仿照百度/高德的 3d 地图，目前处于未完成与搁置阶段，你可以通过 [这个链接](https://map-3d-one.vercel.app/) 来查看 demo。

![simple](./image-hosting/simple.png)

<br />

# 运行

依次执行下述命令，来运行这个项目：

1. `npm install`
2. `npm run start`
